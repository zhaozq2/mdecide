# 多轮打分筛选决策系统mdecide

## 项目简介

基于"多轮打分筛选法"的决策辅助工具，帮助用户通过结构化、可重复的评估流程做出更理性的选择。系统支持创建任意决策场景，对筛选条件进行重要性（1-5分）和必要性（0/1）打分，并通过多轮权重调整，最终生成带总结的决策结果。

**核心决策逻辑**：
1. 罗列筛选条件
2. 重要性打分（1-5）→ 排序
3. 必要性打分（0/1）
4. 30分钟内得出初步结果
5. 第二天/多轮重新打分，获得更稳定结论

## 功能需求

### 核心功能

1. **决策管理**
   - 创建决策（标题、描述、预期轮次数）
   - 编辑/删除决策
   - 查看决策列表

2. **条件管理**
   - 为决策添加多个筛选条件
   - 支持条件增删改

3. **打分流程**
   - **重要性打分**：每个条件 1~5 分（整数），提交后自动按重要性降序排列
   - **必要性打分**：每个条件 0（不必要）或 1（必要），独立于重要性
   - 支持分轮次打分（第1轮、第2轮……）

4. **筛选结果**
   - 根据必要性过滤：必要条件（=1）必须全部满足
   - 按重要性总分排序（可扩展加权和，当前版本仅展示重要性与必要性汇总）
   - 生成简单总结描述（例如：“第2轮筛选后，【条件A】为必要，【条件B】重要性最高，建议优先考虑…”）
   - 支持查看历史轮次结果对比

5. **多轮机制**
   - 同一决策可开始新的一轮打分
   - 后端保存每轮的打分记录及结果快照
   - 前端可对比不同轮次的结论

### 附加要求
- 必要性打分（0/1）采用开关或单选按钮，视觉清晰
- 重要性打分支持滑块或5星/数字按钮
- 结果页展示“最终建议” + 条件排名
